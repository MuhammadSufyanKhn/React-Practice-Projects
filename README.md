# 🚀 React Practice Projects

This repository contains a collection of **React.js practice projects** built to improve frontend development skills, understand core concepts, and experiment with modern tools.

---

## 📌 About

The goal of this repository is to:

- Practice **React fundamentals**
- Build small to medium-sized projects
- Understand **state management, hooks, and API handling**
- Improve UI/UX with **Tailwind CSS**
- Work with real-world data using APIs
- Experiment with libraries for **file handling, drag-and-drop, and PDF manipulation**

---

## 🗂️ Projects

### 📄 PDF Merger
A fully client-side PDF tool. Upload multiple PDFs, reorder pages via drag & drop, crop individual pages, and merge everything into one downloadable PDF — no server involved.

> **Tech:** pdf-lib · pdfjs-dist · @dnd-kit · react-image-crop

---

### 🖼️ Image Gallery
A responsive image gallery built with React.

> **Tech:** React.js · Tailwind CSS

---

### 💼 Job Opening UI
A job portal UI mini project showcasing job listings and opening cards.

> **Tech:** React.js · Tailwind CSS

---

## 🛠️ Tech Stack

- ⚛️ React.js (Vite / Create React App)
- 🎨 Tailwind CSS
- 🌐 Axios (API calls)
- 📦 JavaScript (ES6+)
- 📄 pdf-lib, pdfjs-dist (PDF projects)
- 🖱️ @dnd-kit (drag-and-drop)

## ⚙️ Getting Started

### Prerequisites
Make sure you have **Node.js** installed → [nodejs.org](https://nodejs.org) (LTS version)

### 1. Clone the repository
```bash
git clone https://github.com/MuhammadSufyanKhn/react-practice-projects.git
```

### 2. Navigate to a project folder
```bash
cd react-practice-projects/pdf-merger
# or
cd react-practice-projects/Image-Gallery
# or
cd react-practice-projects/Job-Opening-Ui
```

### 3. Install dependencies
```bash
npm install
```

### 4. Run the development server
```bash
npm start
# or
npm run dev
```
