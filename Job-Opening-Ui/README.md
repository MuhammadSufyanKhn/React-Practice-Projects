# JobPortal React UI

A modern, responsive Job Board interface built with **React** and **Vite**. This project features a clean, card-based UI design showcasing various tech job openings with real-time data mapping and a polished aesthetic.

## 🚀 Features
* **Dynamic Data Rendering:** Uses JavaScript `.map()` to render job listings from a data array.
* **Reusable Components:** Built with a modular `Card` component for high maintainability.
* **Modern UI/UX:** Clean, flat design focusing on typography and spacing.
* **Lucide Icons:** Integrated `lucide-react` for professional iconography.
* **Responsive Layout:** CSS Flexbox system that organizes cards into a wrapping grid.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Build Tool:** Vite
* **Styling:** Custom CSS3
* **Icons:** Lucide React
